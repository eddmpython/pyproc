# attempts: boundaryRadius

**가설.** 경계("다시부팅해서 도달하는 힙")의 동일성에는 **반경**이 있다. 그 반경 안에서만
`상태 = 경계 + 델타`의 뺄셈이 성립한다. 반경이 어디까지인지 아무도 재지 않았다.

**설계 정본**은 [mainPlan/boundary-radius](../../../mainPlan/boundary-radius/README.md)다.
여기는 그 1단계(측정 장치)를 브라우저에서 돌리는 자리다.

## 왜 이 캠페인이 따로 있나

`pythonMachine/bootDeterminismProbe.html`은 **같은 탭에서 두 번 부팅**해서 180p -> 0p를 얻었다
(`let a = await boot(), b = await boot()`). 그건 pythonMachine의 닫힌 결과이고 건드리지 않는다.

이 캠페인이 묻는 것은 다른 질문이다: **그 0이 탭을 넘나, 기기를 넘나, 빌드를 넘나.** 축이
다르면 실험이 다르다. 우리 자신의 실측이 이미 "한 브라우저 안에서도 메인과 워커는 바이트가
다르다"(`src/processOs/worker.js:10`)를 갖고 있으므로, 반경이 좁을 가능성이 실재한다.

## 측정 축

| 축 | 상태 | probe |
|---|---|---|
| 부팅 대 부팅 (같은 탭) | 실측됨: 엔트로피 고정 후 0p | pythonMachine(기존) |
| 워커 대 워커 (같은 탭) | 실측됨: 바이트 동일 8/8 | pythonMachine(기존) |
| 메인 대 워커 (같은 탭) | 실측됨: **다름** | `worker.js:10` 주석 |
| **탭 대 탭 (같은 기기)** | **미측정** | `tabRadiusProbe.html` |
| **기기 대 기기** | **미측정** | `deviceRadiusProbe.html` + CI |
| **빌드 대 빌드** | **미측정** | 위와 같은 장치, CI가 두 Chromium |

## 장치

`boundaryChallenge.js` 하나가 셋을 한다. `guix challenge`가 파일에 하는 것을 부팅 후 힙에 한다.

1. **매니페스트** - "같은 조건"의 정의. 지금 코드 어디에도 이 정의가 없다.
2. **벡터** - 부팅 -> 페이지 해시 (`MemoryCapability.pageHashes()`, 페이지당 64비트).
3. **다이제스트** - 매니페스트 + 벡터의 sha256. **경계의 content address.**

두 참가자는 **hex 문자열 하나**를 비교한다. 같으면 경계가 같고, 다르면 벡터를 대조해 **어느
페이지인지** 뽑는다. 경계 자체(40MB)는 한 번도 전송하지 않는다. 벡터는 약 5KB다.

## 두 번째 기기를 사지 않는다

**CI 러너가 두 번째 기기다.** 다른 CPU, 다른 커널, 종종 다른 Chromium 빌드. 벡터를 커밋하고
CI가 재계산해 대조하면 그게 기기 대 기기 측정이고 비용이 0이다.

## 졸업 게이트

`src/`로 승격하려면 전부 필요하다.

1. **탭 대 탭과 기기 대 기기의 상이 페이지 수가 숫자로 있다.** 0이든 아니든 산출물이다.
2. **0이 아닌 축은 발산 출처가 특정된다.** `diffIndices`가 이름을 얻는다(주소/패딩/타임스탬프/
   hiwire 슬롯 중 무엇인지).
3. **측정된 것만 게이트가 된다.** 0인 축만. 결과를 모르는 것을 게이트로 만들지 않는다
   (신설 게이트는 음성 시험으로 이빨을 증명하고서야 게이트다).
4. **반경이 계약으로 문서에 박힌다.** 능력 매트릭스가 한정을 말한다.
5. 매니페스트 정의가 **덕지덕지 없이** 선다. 플래그 누적으로 "같음"을 만들지 않는다.

## 하지 않는 것

- **"경계는 공공재"를 주장하지 않는다.** JVM CDS(2019)가 배송 중이고 Cloudflare workerd가
  Pyodide 위에서 baseline + dedicated stacked snapshot으로 출시했다. 폐기 근거는 원장에 있다.
- **속도를 주장하지 않는다.** 이 캠페인은 시간을 재지 않는다. 바이트만 잰다.
